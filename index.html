Motorcycle Telemetry & Lean‑Angle System • 2024–present

Problem: Affordable, rider‑friendly analytics for lean angle, braking, throttle, tire/road temps.

Approach: ICM‑42688‑V + sensor fusion; custom PCB + firmware; CAN/GPS integration; compact TFT UI.

How: Rapid prototypes via 3D print + CNC; Python tools for logging; calibration rigs; on‑bike tests.
